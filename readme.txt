== Description ==

#### Katorymnd Contact form for WordPress

*With this plugin, you will allow your website visitors to send a contact message with ease.*

Katorymnd Contact Form is a great deal in confirming the security and securing your contact details. The form has inbuilt CAPTCHA and also validates your contact page URL before any message is sent.

[youtube https://www.youtube.com/watch?v=MZEjlTbDznc]

#### Major features include

* Automatically prompt any user to pass a security check before submitting the form.
* Your WordPress contact form URL page is securely verified to stop any user or client from modifying the contact page URL address.
* Secure email submission.
* You're free to choose, either to use your default WordPress admin email or assign a new custom email address to receive your website contact messages.
* Place **`<?php do_shortcode('katorymnd_lob_contact_form'); ?>`** in your templates

== Installation ==

1. In your WordPress admin panel, go to **Plugins > Add New**.
2. Search for **Katorymnd Contact Form** using the search bar.
3. Click **Install Now** next to the Katorymnd Contact Form plugin.
4. After the plugin is installed, click **Activate** to enable it on your site.
5. To configure the plugin, navigate to **Settings > Katorymnd Contact Form**. Here you can access and adjust the Katorymnd WordPress contact form settings.

== Frequently Asked Questions ==

#### How to display a form in your contact page?
